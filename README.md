This is the source code for the Coderave website.

## Development

1. Clone the repository:
    ```bash
    git clone https://github.com/coderavehq/coderave-website.git
    cd coderave-website
    ```

2. Install the dependencies:
    ```bash
    pnpm install
    ```

3. Start the development server:
    ```bash
    pnpm dev
    ```
